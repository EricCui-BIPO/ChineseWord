# 如何扩展汉字数据到3000个

## 📝 当前状态

目前应用包含20个示例汉字。要扩展到3000个汉字，您有以下几种方法：

## 🚀 方法一：手动添加（推荐用于小批量）

直接编辑 `src/data/words.json` 文件，按照以下格式添加：

```json
{
  "id": "word-021",
  "character": "新",
  "pinyin": "xīn",
  "meaning": "刚出现或刚经验到的",
  "words": ["新年", "新鲜", "新闻"],
  "category": "一年级",
  "difficulty": 1
}
```

## 🤖 方法二：使用脚本生成（推荐）

### 步骤1：安装依赖

如果需要使用拼音转换库，可以安装：

```bash
npm install pinyin-pro --save
```

### 步骤2：创建生成脚本

我已经创建了一个脚本模板：`scripts/generate-3000-words.js`

您可以：
1. 从在线字典API获取汉字数据
2. 使用拼音转换库自动生成拼音
3. 从字典API获取意思和组词

### 步骤3：运行脚本

```bash
node scripts/generate-3000-words.js
```

## 📚 方法三：使用数据源

### 推荐数据源：

1. **汉字字典API**
   - 新华字典API
   - 在线汉字字典
   - 百度汉语API

2. **数据文件**
   - 《现代汉语常用字表》
   - 《通用规范汉字表》
   - 常用汉字数据库

## 📋 数据结构说明

每个汉字需要包含以下字段：

- `id`: 唯一标识（如：word-0001）
- `character`: 汉字字符
- `pinyin`: 拼音（带声调，如：zhōng）
- `meaning`: 意思解释
- `words`: 组词示例数组（可选）
- `category`: 分类（如：一年级、二年级等）
- `difficulty`: 难度等级（1-5）

## 🎯 快速扩展建议

1. **使用现有API**：
   - 使用汉字字典API批量获取数据
   - 使用爬虫从在线字典获取

2. **分批添加**：
   - 先添加常用字（前500个）
   - 再逐步扩展到3000个

3. **使用数据库**：
   - 将汉字数据存储在数据库中
   - 通过API动态加载

## ⚠️ 注意事项

- 确保拼音准确性
- 确保意思解释准确
- 合理安排分类和难度
- 注意数据文件大小（JSON文件过大可能影响加载性能）
