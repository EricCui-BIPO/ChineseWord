# 快速启动指南

## 🚀 三步启动应用

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 打开浏览器
访问 `http://localhost:3000`

## 📱 使用说明

### 首页功能
- 选择学习模式（练习/测试/复习）
- 查看学习统计
- 选择汉字分类

### 练习模式
- 点击卡片翻转查看详情
- 使用按钮切换上一个/下一个
- 标记已学习或已掌握

### 测试模式
- 随机显示汉字进行测试
- 点击"查看答案"翻转卡片
- 标记答对或答错
- 查看测试统计

### 复习模式
- 自动筛选需要复习的汉字
- 重点复习未掌握或困难的汉字

## 🎨 主题特色

- 艾莎公主主题配色（冰雪蓝、淡紫色）
- 卡片翻转动画
- 响应式设计，支持手机和电脑
- 优雅的UI界面

## 💾 数据说明

- 所有学习进度自动保存到浏览器本地存储
- 预置20个常用汉字示例
- 可以修改 `src/data/words.json` 添加更多汉字

## 🛠 常见问题

**Q: 如何添加更多汉字？**
A: 编辑 `src/data/words.json` 文件，按照现有格式添加汉字数据。

**Q: 如何清除学习进度？**
A: 在浏览器开发者工具中清除 localStorage，或等待后续版本添加清除功能。

**Q: 如何部署到网上？**
A: 运行 `npm run build` 构建项目，然后将 `dist` 目录部署到任何静态网站托管服务（如 Vercel、Netlify、GitHub Pages）。
