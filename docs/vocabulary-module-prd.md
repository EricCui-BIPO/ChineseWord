# 词语学习模块 - 产品需求文档 (PRD)

**项目**: ChineseWord - 词语学习模块
**版本**: 1.0
**日期**: 2025-11-07
**状态**: BMAD 框架设计中

---

## 📋 执行总结

词语学习模块是 ChineseWord 学习应用的第三个核心学习功能，专注于帮助 6-12 岁小学生通过**游戏化学习方式**掌握约 1,000 个小学常用词语。

本模块将与现有的**汉字学习**（汉语文字学习）和**成语学习**（短语和习语学习）相辅相成，提供更加全面的中文学习体验。

### 核心承诺
- ✅ **包含 1,000+ 小学常用词语**（按年级和主题组织）
- ✅ **游戏化学习体验**（不仅仅是传统复习）
- ✅ **自动进度追踪**（本地存储，随时随地学习）
- ✅ **Elsa 冰雪主题**（与现有应用保持视觉一致性）
- ✅ **响应式设计**（完美适配手机、平板、桌面）

## 👥 用户画像

### 主要用户：小学生（6-12岁）
- **年龄**: 一至六年级（约 6-12 岁）
- **学习水平**: 初级到中级汉语水平
- **技术素养**: 熟悉手机/平板操作
- **学习动机**: 学校作业辅助、兴趣学习、家长督导

### 次要用户：家长和教师
- **家长**: 监督孩子学习进度
- **教师**: 补充课堂教材，分配作业

### 用户需求

**小学生**:
- 学习应该**有趣且互动**，而不是枯燥的闪卡
- 需要**明确的进度反馈**（分数、成就、进度条）
- 希望能**快速学习**（碎片化时间）
- 想要**视觉吸引**（色彩、动画、奖励）

**家长**:
- 想要看到**学习进度**（学了多少个词）
- 需要**学习内容可信**（真实词汇，与教材对应）
- 期望**使用时间合理**（不上瘾，但足够有吸引力）

## 🎮 功能需求

### 1. 词语库管理

#### 1.1 数据规模
- **总词汇量**: 1,000 个常用词语
- **覆盖范围**: 小学一年级至六年级
- **分布**: 每年级约 150-200 个词

#### 1.2 词语组织方式

**方式 1: 按年级分类**
```
一年级词汇 (学习简单日常词语)
  - 150-170 个词
  - 难度: 简单 (拼音为主)

二年级词汇
  - 160-180 个词
  - 难度: 简单-中等

...依此类推...

六年级词汇
  - 180-200 个词
  - 难度: 中等-困难
```

**方式 2: 按主题分类**
```
动物 (50+) - 狐狸, 蝴蝶, 老鼠, ...
植物 (40+) - 菊花, 柳树, 竹子, ...
食物 (50+) - 苹果, 面条, 米饭, ...
日常用语 (100+) - 开心, 勇敢, 聪明, ...
学校相关 (60+) - 老师, 作业, 课本, ...
家庭相关 (50+) - 爸爸, 妈妈, 弟弟, ...
自然现象 (50+) - 下雪, 下雨, 天空, ...
颜色/形容词 (100+) - 红色, 漂亮, 软, ...
... 更多主题 ...
```

**方式 3: 按难度分级**
```
简单 (300+) - 常见日常词, 高频词汇
中等 (400+) - 学校常用词, 较复杂概念
困难 (300+) - 年级词汇, 细致描述词
```

#### 1.3 词语数据结构

每个词语条目包含：
```json
{
  "id": "vocab-0001",
  "word": "蝴蝶",
  "pinyin": "hú dié",
  "meaning": "一种昆虫，有彩色的翅膀，喜欢在花朵上飞舞",
  "exampleSentence": "花园里飞着许多美丽的蝴蝶。",
  "synonym": ["彩蝶"],
  "antonym": [],
  "category": "动物",
  "grade": 2,
  "difficulty": 1,
  "themes": ["动物", "昆虫", "自然"],
  "imageUrl": "assets/images/vocab/butterfly.png"
}
```

### 2. 学习模式

#### 2.1 游戏化学习模式（主模式）
> 类似打地鼠游戏，但以词语学习为主题

**游戏玩法选项**:

**选项 A: 词语消消乐** (推荐首选)
- **界面**: 9 宫格，每个格子显示一个词语
- **题目**: 在上方显示：
  - 释义提示：「一种昆虫，有彩色的翅膀」
  - 或拼音提示：「hú dié」
  - 或例句提示：「花园里飞着许多美丽的___」
  - 三种类型随机轮换
- **玩法**: 玩家点击对应的词语卡片
- **计分**:
  - 正确: +5-10 分（根据难度）
  - 错误: -1 分
  - 连续答对 3 个: +20 分奖励
- **时间**: 60 秒计时，越快完成越高分
- **难度递进**:
  - 0-20 秒: 简单词汇
  - 20-40 秒: 混合词汇
  - 40-60 秒: 困难词汇

**选项 B: 拼音配对** (备选)
- 左侧显示词语，右侧显示拼音
- 通过拖拽或点击进行配对
- 配对完成后自动检验并计分

**选项 C: 主题闯关** (未来扩展)
- 按主题分关卡
- 每关 10-15 个词语
- 三星评分系统

#### 2.2 传统学习模式（可选）
- **学习模式**: 顺序翻页学习词语及其含义
- **复习模式**: 系统推荐需要复习的词语
- **详情页**: 查看完整的释义、例句、近义词等

### 3. 进度追踪

#### 3.1 学习状态
每个词语有三种状态：
- **未学习** (默认状态)
- **已学习** (用户标记为学过)
- **已掌握** (用户标记为完全掌握)

#### 3.2 个人统计
- 总词汇数: 1,000
- 已学习: X 个 (X%)
- 已掌握: Y 个 (Y%)
- 本周学习: Z 个
- 连续学习天数

#### 3.3 分数追踪
- **当日最高分**: 本天游戏最高分
- **总分**: 累计分数
- **排行榜**: 可选，未来扩展（周排行、月排行）

### 4. 个性化学习路径

#### 4.1 推荐策略
- **新用户**: 从一年级简单词汇开始
- **进阶学习者**: 根据掌握度推荐下一级词汇
- **复习**: 推荐 7 天未复习且未完全掌握的词汇

#### 4.2 难度自适应
- 系统根据用户答题正确率自动调整难度
- 正确率 > 80%: 显示更难的词汇
- 正确率 < 50%: 返回更简单的词汇

### 6. 内容需求

#### 6.1 词语选择标准
- **来源**: 小学语文教学大纲（中国）
- **频度**: 常用词汇（教科书出现频率高）
- **年级对应**: 按教学阶段合理分配
- **多样性**: 涵盖动物、植物、日常、学校、家庭等主题

#### 6.2 数据质量
- **拼音准确**: 使用标准汉语拼音（带声调）
- **释义准确**: 符合小学生理解水平的简明释义
- **例句真实**: 来自小学课本或教学材料的真实例句
- **近义词完整**: 为能提升学习效果的词汇添加近义词

## 🎨 非功能需求

### 1. 性能需求
- **初始加载**: < 2 秒
- **游戏帧率**: 稳定 60 FPS
- **内存占用**: < 100 MB
- **包大小**: +50-80 KB (gzipped)

### 2. 兼容性需求
- **浏览器**: Chrome 90+, Safari 14+, Firefox 88+
- **设备**: iOS 12+, Android 8+
- **屏幕尺寸**: 375px (手机) 到 1920px (桌面)

### 3. 可访问性需求
- **键盘导航**: 全功能支持（用于教室投影）
- **屏幕阅读器**: 基础支持
- **高对比度**: 支持高对比度模式

### 4. 安全性需求
- **数据隐私**: 本地存储，不发送到服务器
- **进度备份**: 支持本地导出（未来扩展）

### 5. 可扩展性需求
- **架构**: 支持未来添加多种游戏模式
- **数据**: 支持扩展到 5000+ 词语
- **主题**: 支持主题切换（节日主题等）

## 📝 附录：关键指标定义

### 学习完成率
```
= (已学习词汇数 + 已掌握词汇数) / 1000
越高越好，目标 > 80%
```

### 掌握度
```
= 已掌握词汇数 / 1000
表示用户真正掌握的词汇比例，目标 > 60%
```

### 游戏参与度
```
= 选择游戏模式的用户 / 总用户
表示用户喜爱游戏化学习的比例，目标 > 50%
```
