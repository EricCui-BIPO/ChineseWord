# 词语学习模块 - UX 设计规格说明

**项目**: ChineseWord - 词语学习模块
**版本**: 1.0
**日期**: 2025-11-07
**设计工具**: Figma (参考), Vue + Tailwind CSS (实现)

## 📱 设计系统

### 色彩体系（Elsa 冰雪主题）

**主色调** (保持与现有应用一致):
- **冰蓝主色**: `#667eea` (按钮、高亮、重点文本)
- **冰蓝浅色**: `#e0e7ff` (背景、辅助元素)
- **紫色点缀**: `#9f7aea` (卡片边框、装饰)
- **白色**: `#ffffff` (卡片背景)
- **淡灰**: `#f7fafc` (背景色)
- **深灰**: `#2d3748` (主文本)

**补色** (游戏反馈):
- **成功绿**: `#48bb78` (正确答案)
- **错误红**: `#f56565` (错误答案)
- **警告黄**: `#ed8936` (注意项)

**渐变色**:
```css
冰雪渐变: linear-gradient(135deg, #667eea 0%, #764ba2 100%)
冰晶渐变: linear-gradient(to right, #e0e7ff 0%, #f7fafc 100%)
```

### 字体系统

**字体栈** (使用现有 Tailwind 配置):
```
标题: 加粗 Serifless (Apple默认字体)
正文: 常规 Serifless
代码: 等宽字体

字号:
- H1 (主标题): 48px (mobile 36px)
- H2 (页面标题): 32px (mobile 24px)
- H3 (卡片标题): 24px (mobile 20px)
- Body Large: 18px
- Body Normal: 16px
- Body Small: 14px
- Caption: 12px
```

### 圆角半径

```css
小: 8px (按钮、输入框)
中: 12px (卡片、模态框)
大: 16px (主要容器)
超大: 20px (特殊装饰)
```

### 间距系统

```
基础单位: 4px (Tailwind 的 spacing scale)

常用间距:
- xs: 4px (细微间距)
- sm: 8px (紧凑间距)
- md: 12px (标准间距)
- lg: 16px (宽松间距)
- xl: 24px (大间距)
- 2xl: 32px (超大间距)
```

### 阴影系统

```css
卡片阴影: 0 4px 12px rgba(102, 126, 234, 0.15)
悬停阴影: 0 8px 24px rgba(102, 126, 234, 0.25)
按钮阴影: 0 2px 8px rgba(0, 0, 0, 0.1)
```

## 🎮 用户旅程

### 新用户旅程
```
1. 打开应用 (Home.vue)
   ↓
2. 看到词语学习卡片
   ↓
3. 点击进入词语模块 (VocabularyHome.vue)
   ↓
4. 选择学习难度和方式
   ↓
5. 开始游戏 (VocabularyGame.vue)
   ↓
6. 完成游戏，查看分数
   ↓
7. 返回主页或继续挑战
```

### 回访用户旅程
```
1. 进入词语模块
   ↓
2. 系统推荐继续学习（基于上次进度）
   ↓
3. 快速开始游戏
   ↓
4. 完成后查看总体进度统计
```

```
│                                           │
└───────────────────────────────────────────┘
```

**设计细节**:

**顶部状态栏**:
- 左: 返回按钮（← 图标）
- 中: 倒计时（⏰ 时间秒数）
- 右: 实时分数（💯 数字）
- 背景: 白色，底部淡蓝线条
- 高度: 44px (mobile), 56px (desktop)

**题目区域**:
- 背景: 浅蓝色 (#e0e7ff)
- 文本: 深灰色，左对齐或居中
- 字号: 18px (mobile), 24px (desktop)
- 内边距: 16px (mobile), 24px (desktop)
- 圆角: 12px
- 内容: 动态更新（每答对一题新题目出现）

**答案网格**:
- 布局: 3x3 网格（Mobile）, 3x3 网格（Desktop，可扩展）
- 卡片宽度: (屏幕宽度 - 48px 边距) / 3
- 卡片高度: 等于卡片宽度（正方形）
- 边距: 8px (卡片间)
- 背景: 白色，蓝色边框（2px）
- 文字: 14px (mobile), 16px (desktop), 蓝色，加粗
- 圆角: 8px
- 触碰目标: >= 44x44px

**卡片状态**:
- **默认**: 白色背景，蓝色边框，蓝色文字
- **悬停**: 背景变浅蓝色，阴影增强
- **按下**: 背景变深蓝色，文字白色（视觉反馈）
- **正确**: 背景绿色，勾选图标显示
- **错误**: 背景红色，X 图标显示

**反馈区域**:
- 显示位置: 网格下方或中央弹窗
- 显示时间: 1 秒后消失，加载下一题
- 动画: 淡入-淡出或缩放
- 文本:
  - 正确: "[✓ 正确!] +10 分" (绿色)
  - 错误: "[✗ 错误!] -1 分，正确答案是蝴蝶" (红色)

**连击指示器**:
- 显示位置: 题目区域下方或右上角
- 样式: "连击: X/3 [🔥🔥🔥]"
- 当达到 3 个连击时闪烁并显示 "+20 分奖励"

**响应式设计**:

| 设备类型 | 宽度 | 网格 | 卡片大小 |
|---------|------|------|----------|
| Mobile 竖 | 375px | 3x3 (9) | ~85x85 |
| Tablet 竖 | 640px | 3x4 (12) | ~150x150 |
| Tablet 横 | 1024px | 3x3 (9) | ~200x200 |
| Desktop | 1280px+ | 3x3 (9) | ~300x300 |

## ⌨️ 交互和动画

### 按钮交互

**悬停效果**:
```css
transform: translateY(-2px);
box-shadow: 0 8px 24px rgba(102, 126, 234, 0.25);
transition: all 0.2s ease;
```

**按下效果**:
```css
transform: scale(0.95);
```

**禁用状态**:
```css
opacity: 0.5;
cursor: not-allowed;
```

### 卡片翻转动画

```css
@keyframes flip {
  0% {
    transform: rotateY(0deg);
  }
  100% {
    transform: rotateY(180deg);
  }
}
```

### 答案反馈动画

**正确**:
```css
/* 绿色闪烁 + 放大效果 */
@keyframes correct {
  0% { transform: scale(1); opacity: 1; }
  50% { transform: scale(1.1); }
  100% { transform: scale(1); opacity: 0; }
}
```

**错误**:
```css
/* 红色晃动 + 缩小 */
@keyframes wrong {
  0%, 100% { transform: translateX(0); opacity: 1; }
  25% { transform: translateX(-10px); }
  75% { transform: translateX(10px); }
}
```

### 页面过渡

```css
/* 淡入淡出 */
transition: opacity 0.3s ease;

/* 滑动 */
transition: transform 0.3s ease;
```

### 微交互

**点击涟漪**:
```css
当用户点击卡片时，显示涟漪扩散动画
```

**加载骨架屏**:
```css
灰色脉冲动画，直到内容加载
```
